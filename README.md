# Prolific Rehab Center

A premium, conversion-focused physiotherapy website built with **Next.js 16** (App Router), **React 19**, and **Tailwind CSS v4**. Features a modern healthcare UI, a fully responsive mobile-first design, SEO-friendly metadata, accessible components, and a performance-focused static architecture to deliver an engaging user experience and maximize appointment bookings.

## Tech stack
- Next.js 16 (App Router, Turbopack)
- React 19
- Tailwind CSS v4
- lucide-react icons

## Getting started
```bash
npm install
npm run dev     # start the dev server on http://localhost:3000
npm run build   # production build
npm run start   # serve the production build
```
